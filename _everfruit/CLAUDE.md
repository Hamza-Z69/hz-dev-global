# Everfruit Digital — Rapports clients

## Description
Rapports de performance pour les clients gérés en partenariat avec Everfruit Digital (agence franchise). Chaque rapport est un HTML autonome déployé sur Vercel.

## Clients
- `monceau-fleurs/` — Reporting EVF complet (codir franchise)
- `bodyhit-saintlazare/` — Rapport performance campagne B2C

## Conventions
- Design dark premium, sections dépliables
- Logo Everfruit + logo client en header
- Données vérifiées et auditées avant livraison
- Attribution plateforme (Meta/Google) pour les CPL

## Portail
Les rapports sont référencés dans le portail Meta-Advertising sous l'onglet Everfruit Digital.
