# Meta-Advertising ∞ — Newsletter

## Infos clés
- **Nom** : Meta-Advertising
- **Logo** : symbole ∞ (infini) massif sur gradient bold violet→rose→orange
- **Thème** : Marketing digital (SEO, Ads, IA, social media, lead gen, prospection)
- **Cible** : Les clients finaux de l'agence (commerçants, entreprises) pour qui on gère les campagnes pub et stratégies marketing
- **Fréquence** : Hebdomadaire
- **Numéro actuel** : #44 (dernière édition envoyée le 18 mai 2026)
- **Source articles** : blogdumoderateur.com en priorité, mais aussi siecledigital.fr, codeur.com si BDM manque de sujets pertinents
- **Validation** : L'utilisateur valide le choix des 2 articles à chaque fois

## Workflow par édition
1. Aller sur blogdumoderateur.com, lister les articles récents
2. Proposer une sélection → l'utilisateur choisit 2 articles
3. Fetcher les 2 articles, rédiger les résumés + points clés
4. Fetcher les images directement depuis les articles source (f.hellowork.com/blogdumoderateur/...)
5. Générer 4 tips de la semaine ORIENTÉS CLIENTS (pas techniques/agence)
6. Mettre à jour le numéro d'édition et la date
7. Copier le HTML dans le presse-papier → l'utilisateur colle dans Gmail (Cmd+A, Cmd+C depuis le navigateur, Cmd+V dans Gmail)

## Structure du template
1. **Header** : Gradient bold (violet→rose→orange) + ∞ massif 120px + META-ADVERTISING
2. **Édito** : Accroche contextualisant les 2 articles
3. **Sommaire** : 2 items numérotés
4. **Article 1** : Image + tag + titre + résumé + points clés (pas de lien "lire l'article")
5. **Article 2** : Idem
6. **Tips de la semaine** : 4 tips ORIENTÉS CLIENTS FINAUX (comprendre le digital, optimiser ses pubs, formats, Google Business, réseaux sociaux, avis...)
7. **CTA Accompagnement** : "Une idée ? Une question ?" + bouton "Contactez-nous" (mailto avec sujet pré-rempli)
8. **Footer** : Logo ∞ + réseaux sociaux + désabonnement

## Design
- **Style** : Moderne & Coloré, gradient bold
- **Fond** : Sombre (#08080F / #0F0F1C)
- **Palette gradient** : #1E3A5F → #0D9488 → #10B981 → #34D399 → #F59E0B (changée au #44 pour rafraîchir)
- **Cards** : #161630 avec bordures subtiles
- **Typo** : Arial/Georgia (email-safe)
- **Format** : HTML email-compatible (tables, CSS inline, support Outlook)

## Consignes éditoriales importantes
- Pas de lien "Lire l'article complet"
- Pas de référence au Blog du Modérateur comme source
- Pas de section "Chiffre de la semaine" (remplacée par Tips)
- **Tips** : toujours orientés clients finaux (pas jargon agence). Exemples : formats pub, avis Google, horaires de publication, fiche Google Business, comment lire ses stats, etc.
- **Articles** : privilégier ceux qui sont utiles/compréhensibles pour les clients (pas trop technique IA pure). OK si IA mais avec un angle "qu'est-ce que ça change pour mon business"
