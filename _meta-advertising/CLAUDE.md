# Meta-Advertising — Portail interne

## Description
Portail interne de l'agence Meta-Advertising. Regroupe tous les clients avec liens vers leurs dashboards et rapports. Design manga/anime.

## Fichier principal
- `meta-advertising.html` — le portail (tableau CLIENTS en JS)

## Ajouter un client
Dans le tableau `const CLIENTS = [...]` de meta-advertising.html, ajouter un objet avec : id, name, sector, status, logo, color, links[].

## Sous-dossiers
- `audit-rnpc/` — Audit RNPC
- `google-ads-agent/` — Agent Google Ads
- `meta-ads-agent/` — Agent Meta Ads
- `schmidt-budgets/` — Suivi budgets Schmidt
