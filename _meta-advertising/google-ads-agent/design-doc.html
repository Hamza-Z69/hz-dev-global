Google Ads Agent — Design Documentation

Meta-advertising · Internal Tool · Version 1.0 · March 2026

1. Overview

Google Ads Agent is an internal command-line tool built by Meta-advertising to manage Google Ads campaigns for our clients. The tool provides a unified interface to create, manage, and monitor campaigns across multiple client accounts.

Purpose: Streamline Google Ads campaign management for our agency team by automating repetitive tasks (campaign creation, keyword management, reporting) through the Google Ads API.

4. Features & API Usage

4.1 Campaign Management

  • Create campaigns (Search, Display, Performance Max, Video, Demand Gen)
  • Set daily budgets with automatic budget resource creation
  • Pause, activate, or remove campaigns
  • List all campaigns with status and budget information

4.2 Ad Group & Keyword Management

  • Create ad groups within campaigns with optional CPC bids
  • Add keywords with match type control (Broad, Phrase, Exact)
  • Add negative keywords at campaign level
  • List and monitor keyword performance

4.3 Ad Creation

  • Create Responsive Search Ads (RSA) with multiple headlines and descriptions
  • Set final URLs and call-to-action
  • Pause or remove underperforming ads

4.4 Reporting & Analytics

  • Account-level performance metrics (impressions, clicks, CTR, CPC, conversions)
  • Campaign-level comparison reports
  • Daily breakdown reports per campaign
  • Keyword-level performance reports (top 50)
  • Configurable date ranges (7, 14, 30 days)

7. Security & Data Handling

  • Authentication: OAuth 2.0 with refresh tokens stored locally in environment variables (.env file, git-ignored)
  • Access control: Internal use only — restricted to Meta-advertising employees
  • Data storage: No customer data is stored locally; all data is read in real-time from the Google Ads API
  • Credentials: Developer token, client ID/secret, and refresh tokens are never committed to version control
  • Scope: Only requests the https://www.googleapis.com/auth/adwords OAuth scope

8. Compliance

  • The tool complies with Google Ads API Terms of Service
  • No automated bidding beyond what Google Ads natively provides
  • No scraping or unauthorized data collection
  • All API calls respect rate limits and best practices
  • GDPR compliant — no personal data is stored outside Google's infrastructure
